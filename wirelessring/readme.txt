Directory Contents:
pic/    contains the PIC 16F876 Source Code 
test/  contains two demos using the Wireless Ring Protocol
test/pathfollower  it requires some extra stuff to work as a printed track
test/simpledemo  is a simple communication demo with two RCXs exchanging datas

Please note that in both demo, to make the things work, it's necessary to send to RCXs a first IR init message as described in Wireless page on http://erika.sssup.it
